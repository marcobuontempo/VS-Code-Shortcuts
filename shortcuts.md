BASIC EDITING

# Cut Line
( Ctrl+X )
<!-- Cuts an entire line (if nothing is already selected) -->
    1. Press Ctrl+X while text cursor is on this line
    2. ...
    3. ...
    4. ...
    5. Press Ctrl+V while text cursor is on the line below:



# Copy Line
( Ctrl+C )
<!-- Copies an entire line (if nothing is already selected) -->
    1. Press Ctrl+C while text cursor is on this line
    2. ...
    3. ...
    4. ...
    5. Press Ctrl+V while text cursor is on the line below:



# Move Line Up or Down
( Alt+ ↑ / ↓ )
<!-- Cuts an entire line (if nothing is already selected) and moves it up/down 1 line -->
    1. Press Alt+↓ while cursor is on this line
    2. ...
    3. ...
    4. ...
    5. Press Alt+↑ while cursor is on this line



# Copy/Duplicate Line Up or Down
( Shift+Alt+ ↑ / ↓ )
<!-- Copies an entire line (if nothing is already selected) and moves it up/down 1 line -->
    1. Press Shift+Alt+↓ while cursor is on this line
    2. ...
    3. ...
    4. ...
    5. Press Shift+Alt+↑ while cursor is on this line


# Jump to matching bracket
( Ctrl+Shift+\ )
<!-- Moves cursor to the matching open/close bracket -->
    Example Code Block {
        1. ...
        2. ...
        3. Press Ctrl+Shift+\ here
        4. ...
        5. ...
    }


# Indent / Outdent Cursor
( Tab / Shift+Tab )
<!-- Indents or Outdents a line of code -->
    1. Press Tab while text cursor is in the middle of this line
    2. Press Shift+Tab while text cursor is in the middle of this line


# Indent / Outdent Line
( Ctrl+ ] / [ )
<!-- Indents or Outdents a line of code -->
    1. Press Ctrl+] while text cursor is in the middle of this line
    2. Press Ctrl+[ while text cursor is in the middle of this line


# Fold(collapse) or Unfold(uncollapse) a Region
( Ctrl+Shift+[  /  Ctrl+Shift+])
<!-- Collapses/Uncollapses a Region -->
    1. Press Ctrl+Shift+[, then press Ctrl+shift+]


# Toggle Line Comment
( Ctrl+/ )
    Toggles a line of code to comment/uncomment. Press Ctrl+/ while text cursor is on this line



SEARCH AND REPLACE

# Find
( Ctrl+F )
<!-- Finds search term in document -->
    1. Press Ctrl+F to search this document for "Hello World"

# Replace
( Ctrl+H )
<!-- Finds and replaces search term in document -->
    1. Press Ctrl+H to search this document for "Hello World" and change to "Hello World"

DISPLAY/NAVIGATION

# Zen Mode
( Ctrl+K, Z )
<!-- Enters fullscreen Zen mode without distractions -->
    1. Press Ctrl+K
    2. Press Z
    3. Press Esc Esc to exit!

# Show Terminal
( Ctrl + ` )
<!-- Shows/Hides terminal -->
    1. Press Ctrl+` to show terminal window

# Create New Terminal
( Ctrl+Shift+` )
<!-- Creates a new terminal -->
    1. Press Ctrl+Shift+` to create new terminal

# Fast Scroll
( Alt + scrollwheel)
<!-- Increases scroll speed by 5x -->
    1. Hold Alt+scrollwheel ↑ / ↓

# Quick Open, Go to File...
<!-- Quickly navigate to specific files within the current project -->
( Ctrl+P )

# Toggle Explorer Focus
( Ctrl+Shift+E )
<!-- Toggles focus between the explorer sidebar and the editor -->
    1. Press Ctrl+Shift+E
    2. Press ↑ / ↓ to select "shortcut.js" file
    3. Press Enter
